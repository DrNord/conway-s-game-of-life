The Conway's Game of Life

by Dr. Nord

(v2.0.0, Apr 23, 2022)

Watch primitive cells evolve in their primitive universe

The famous Game of Life is a cellular automaton invented in 1970 by John Horton Conway - the British mathematician.

It is a zero-player game - the initial state fully determines the game evolution. No further input is required.

Nevertheless, the game realization includes advanced controls:

  • drag-n-drop any picture (jpeg, png) into the game field to see it evolve;
  • use the mouse wheel to scale the game field;
  • click-n-move the mouse to navigate through the zoomed game field;
  • see below for more.

Using a touchscreen device (smartphone or tablet) may improve your gaming experience:

  • swipe (using 1 or 2 fingers) to navigate through the zoomed game field;
  • pinch/zoom (using 2 fingers) to scale the game field.

Switch to edit mode and use your mouse or touchscreen to:

  • - add cells;
  • - delete cells;
  • - invert cells;
  • / - clean/fill the whole field;
  • - evolve once (single tap) or continuously (tap-and-move), try when paused.

The sliders button is located in the lowest right corner and opens additional parameters:

  • - the HUD info switch, which displays the current game information;
  • , - the Save/Load buttons, you may save your current game state as an image and choose any picture (jpeg, png) to load - the game will interpret its pixels as dead or alive cells dependent on their brightness;
  • /, /, / - these sliders allow you to scale the field, change sound volume and simulation speed.

The basic game interface also includes:

  • / - the Play/Pause button;
  • - evolve manually by a single click;
  • / - expand/compress the game area by hiding/showing menu bars;
  • - reveal the menu (this button appears in the top right corner when the game field is expanded);
  • - spawn cells randomly;
  • - resize the game field to fit the available area and spawn cells randomly.

The footer menu consists of:

  • - this info;
  • - settings button, where you can manually set game field size, cells spawn probability and simulation speed slider limits;
  • , /, / - sound mute switch, full window expand/compress switch, Day/Night mode switch.

Have fun!
